# Termochimie - Efect Termic

![Termochimie preview](static/design/teorie_ilustratii/caldura_1350_450.png)

Un simulator interactiv de termochimie, facut cu SvelteKit, care explica efectul termic al reactiilor chimice clar si vizual.

Live: [termochimie.bylazar.com](https://termochimie.bylazar.com)

## Showcase

[Vezi showcase-ul](https://youtu.be/-R8AVDQXNUI)

## Pornire rapida

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
bun run preview
```
